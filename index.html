Falta conectar la base de datos

Esta página guarda todo en Firebase (gratis). Sigue estos pasos una sola vez:

  1. Entra a console.firebase.google.com y crea un proyecto nuevo.
  2. Databases & Storage → Firestore → crear base de datos (modo producción).
  3. Security → Authentication → Sign-in method → activa Anónimo.
  4. Configuración del proyecto → Tus apps → Agregar app → Web (</>) → copia el objeto firebaseConfig.
  5. Pega esos valores aquí mismo, al inicio del bloque <script>, donde dice firebaseConfig.
  6. En Firestore → Reglas, pega las reglas de seguridad que te di en el chat.

Cuando termines, guarda el archivo y recarga esta página.
